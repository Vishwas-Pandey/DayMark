DayMark is a full-stack productivity and habit tracking application designed to help users manage tasks, habits, and routines through a secure and scalable system.

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: JWT, bcrypt
- Deployment: Vercel

## Features
- Secure user authentication with JWT and encrypted password storage
- Task, habit, and routine management
- RESTful APIs for user and activity management
- Protected routes and role-based access
- Clean separation of frontend and backend services

## Architecture Highlights
- REST API-based backend with modular route and controller structure
- MongoDB schemas with indexed queries for efficient data access
- Environment-based configuration for local and production deployments
- CORS-safe backend setup

## Getting Started
1. Clone the repository
2. Install dependencies using `npm install`
3. Configure environment variables
4. Run the application using `npm run dev`

- Building secure authentication systems
- Designing scalable REST APIs
